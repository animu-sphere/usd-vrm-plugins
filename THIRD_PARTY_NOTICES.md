### JSON parsing

VRM extension JSON is parsed with OpenUSD's own `pxr/base/js`, so there is **no**
separate JSON library dependency.

### Test assets

generated by `plugins/usdVrmFileFormat/tools/generate_fixtures.py` and contain no
third-party art; they are covered by this repository's Apache-2.0 license.

`vrm_test_data` directory) are **not** part of this repository and are not
redistributed here. Respect each avatar's own VRM `meta` license terms.
